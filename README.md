# Roll a Ball

Proyecto realizado en Unity que consiste en un juego donde el jugador controla una bola que debe moverse por un escenario, esquivar obstáculos, recoger ítems y enfrentarse a un enemigo que la persigue.

## Comenzando

Estas instrucciones te permitirán obtener una copia del proyecto y ejecutarlo en tu máquina local para propósitos de desarrollo y pruebas.  

### Requisitos previos

Para ejecutar este proyecto necesitarás tener instalado:

- [Unity Hub 3.14.3](https://unity.com/download) - Para gestionar el proyecto de Unity.  
- [Visual Studio Code](https://code.visualstudio.com/) - Para editar scripts y depurar código.  

### Instalación

Para ejecutar el proyecto en tu máquina local, sigue estos pasos:

1. Abre Unity Hub.
2. Ve a la sección **Projects** y selecciona **Add** para agregar el proyecto descargado.
3. Navega hasta la carpeta del proyecto y selecciónala.
4. Abre el proyecto con la versión de Unity 3.14.3 indicada en los requisitos previos.
5. Una vez abierto, podrás ejecutar el juego desde el editor haciendo clic en el botón **Play**.

## Ejecutando las pruebas

Para probar el juego, simplemente ejecuta el proyecto desde Unity:

### Prueba del juego

1. Abre el proyecto en Unity.
2. Haz clic en el botón **Play** en el editor.
3. Controla la bola y verifica que pueda moverse, recoger ítems, esquivar obstáculos y que el enemigo la persiga.

### Prueba de estilo

No se han implementado pruebas automáticas de código. Se recomienda seguir buenas prácticas de Unity y C# al modificar los scripts.

## Despliegue

Para desplegar el juego en un sistema en vivo:

1. Abre el proyecto en Unity.
2. Ve a **File > Build Settings**.
3. Selecciona la plataforma de destino (PC, Mac, Web, etc.).
4. Configura las opciones de construcción según tus necesidades.
5. Haz clic en **Build** para generar la versión ejecutable del juego.
